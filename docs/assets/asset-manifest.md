# Demo Asset Manifest

This directory is intentionally committed without fake screenshots or generated GIFs.
Use these stable filenames when adding real assets for the RepoLens AI portfolio demo.

## Expected Filenames

- `docs/assets/repolens-ui-home.png`
  - Streamlit home screen with repo URL input, mode selector, and API base URL caption
- `docs/assets/repolens-trace-summary.png`
  - Ask result screen showing answer, citations, confidence, and the trace summary panel
- `docs/assets/repolens-release-mode.gif`
  - Short workflow recording that ingests a repo and asks a release-mode question

## Capture Guidance

- Keep repository URLs and citations readable.
- Show real traces and diagnostics, not mocked UI output.
- Prefer PNG for screenshots and GIF or MP4 for a short release-mode walkthrough.
